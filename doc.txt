Метод findMax находит максимальное значение в переданном списке чисел
Для этого он:
1. Принимает вектор целых чисел (const vector<int>&)
2. Если вектор пустой, выбрасывает исключение invalid_argument
3. Проходится по всем элементам вектора с помощью индексов, используя линейный поиск для нахождения максимума
4. Возвращает целое число (int)

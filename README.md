# Project №1 - Найти минимум

## Метод: findMin
Этот метод находит минимальное значение в переданном списке.

### Функциональность:
1. Находит минимальное значение в векторе
2. Обрабатывает пользовательский ввод элементов массива
3. Проверяет валидность входных данных для пустых векторов
4. Обрабатывает исключения для неверного ввода

### Параметры:
- `numbers` - Константная ссылка на вектор целых чисел (const vector<int>&)

### Возвращает:
- Минимальное значение из вектора (int)

### Исключения:
- `invalid_argument` если вектор пустой

### Использование:
Программа запрашивает у пользователя:
1. Количество элементов
2. Каждый элемент по отдельности
3. Выводит минимальное значение

### Пример:
```cpp
vector<int> numbers = {1, 5, 3, 9, 2};
int max = findMin(numbers); // возвращает 1

# Project №1 - Найти максимум

## Метод: findMax
Этот метод находит максимальное значение в переданном списке.

### Принцип работы:
1. Принимает вектор целых чисел
2. Если вектор пустой, выбрасывает исключение invalid_argument
3. Использует линейный поиск для нахождения максимума, проходясь по всем элементам вектора с помощью индексов
4. Возвращает целое число int
